字符编码 ascll 计算机只认识二进制，美国人发明的，大小写字母，数字，一些特殊字符 127个，1字节 可以表示，世界各国文字不同，有很多不同的字符 各国标准不同难免冲突，所以要统一，unicode，通常有2个字节表示 偏僻的字符可能用到4个字节，虽然unicode多个字节可以统一， 但是如果全部是英文，用多个字符又很浪费，所有就有了“可变长编码” UTF-8（1-6个字节） 计算机内存中，统一使用unicode编码，读取出来或保存到硬盘就转换成utf-8编码
